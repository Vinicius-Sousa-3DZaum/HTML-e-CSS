Comandos CSS de cor

  • Seletores

    Para fazer algum comando precisa selecionar um bloco

    Chamando blocos como "html" ou "body" colocando "{}"

  • Comandos

    Pode selecionar cores do margin, border, padding, e do centro do texto.

    Comandos como color e background-color

  • Texto

    Existem comandos para demarcar fonte e tamanho do texto

    Comandos como "font-size" que serve para modificar o tamanho do texto e "font-family" que serve pra mudar a fonte do texto.

Variáveis

  • Para criar uma variável precisa criar um novo seletor ":root"

    Para cria a variável no seletor precisa colocar --(dois traços) e colocar o nome da variável (caso precise de espaço entre as palavras use um -)

    • Margem, borda, e padding

  • Para aumentar ou diminuir um espaço pode usar os comandos de margin, border e padding

    "padding: 0.5rem;" determina um espaço de pixels no padding, "margin-left(buttom, top, right): 10px;" determina um espaço de pixels na margem e pode criar uma borda(border) colorida ou animada "border: 7px solid (cor);"

    • Transform

  • Podemos usar o transform para deformar ou mudar a proporção de uma imagem

    "transform: scale(0.7);" pode ser usado para modificar o tamanho da imagem confome muda a escala, "transform: skew(2deg);" deforma a imagem de acordo que muda o skew
